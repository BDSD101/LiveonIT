1) install node.js
2) node --version
3) npm (should come with node) npm --version
4) clone repo (backend part)
5) create a .env file, please see .env.example
6) see new file local.ts
7) install packages, run command "npm install -D dotenv ts-node"
8) useful commands:
# health check and to run locally: 
npm run local

# get users
npm run local:users

# search
npm run local:search

# create user
npm run local:post
